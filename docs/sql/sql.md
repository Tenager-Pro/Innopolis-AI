# Документация SQL запросов

## drop_layers.sql

Данный код выполняет удаление (DROP) SQL-представлений (VIEWS), связанных с различными слоями обработки данных, если они существуют в базе данных.

## drop_tables.sql

Данный код выполняет удаление (DROP) физических таблиц (TABLES), связанных с различными слоями данных в хранилище данных.

## layers.sql

Данный код создает или заменяет SQL-представления (VIEWS) для различных слоев обработки данных и добавляет документационные комментарии.

## queries.sql

Данный код выполняет комплексную проверку качества данных и анализ распределений на разных этапах обработки. Это часть процесса валидации ETL/ELT пайплайна.

## schema.sql

Данный код создает структуру базы данных для анализа клиентов (customer churn analysis), включая саму базу данных и три таблицы, представляющие разные этапы обработки данных.
